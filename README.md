## Royal Paradise Hotel Website ##
---
## Project Overview

The Royal Paradise Hotel Website is a modern, responsive static website developed using HTML5, CSS3, and JavaScript. It provides visitors with complete information about the hotel, luxury rooms, facilities, restaurant, image gallery, and online booking services. The website is designed with a clean and attractive user interface that works seamlessly across mobile, tablet, and desktop devices.

This project was created as part of the **MANTRA 2026 Summer School – Frontend Website Design, SEO, Responsive Design, GitHub & Netlify Hosting Assignment**.
