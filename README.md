## Royal Paradise Hotel Website ##
----
## Project Overview

The Royal Paradise Hotel Website is a responsive static website developed using HTML5, CSS3, and JavaScript. It provides users with information about the hotel, available rooms, facilities, image gallery, and an online room booking form.

This project was created as part of the **MANTRA 2026 Summer School вЂ“ Frontend Website Design, SEO, Responsiveness & Netlify Hosting Assignment**.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

---

## Features

- Responsive design for Mobile, Tablet, and Desktop
- Home page with Hero Banner
- About page with hotel information
- Rooms page with room details and pricing
- Gallery page with hotel images
- Contact page with booking form
- Booking form validation using JavaScript
- Basic SEO implementation
- Modern navigation bar and footer
- Responsive layout using Flexbox and Grid

## SEO Features

- Title tag on every page
- Meta description
- Meta keywords
- One H1 tag per page
- H2 and H3 headings
- Meaningful image alt text
- Simple page names

---

## JavaScript Feature

Booking Form Validation

The booking form validates:

- Name
- Email
- Phone Number
- Check-in Date
- Check-out Date
- Number of Guests

A success message is displayed after successful validation.

---

## Responsive Design

The website is responsive for:

- Mobile
- Tablet
- Desktop

CSS Media Queries, Flexbox, and Grid Layout are used to ensure a user-friendly experience on different screen sizes.
